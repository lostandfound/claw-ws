# 自宅片付けプロジェクト

このフォルダでは、自宅の片付けに関する情報を管理します。

## ファイル構成

- **README.md**: このファイル（プロジェクト概要）
- **areas.md**: エリア別の片付け状況
- **tasks.md**: 片付けタスク一覧
- **items.md**: 処分・整理対象のアイテム記録
- **progress.md**: 進捗記録

## 使い方

Discord #tidy-upチャンネルで片付けに関する相談や記録をしていただければ、amicoがこちらのファイルを更新・管理いたします🌸
