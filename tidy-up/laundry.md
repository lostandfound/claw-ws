# 洗濯物収納システム

## 採用する方法

### ハンガー収納
- 干したままクローゼットへ（畳む工程ゼロ）
- シャツ、ボトムス、アウターなど

### ボックス分け
- 下着・靴下・タオルは畳まずポイッと入れるだけ
- 引き出しやボックスに種類別に分ける

## 必要なもの

### 購入リスト
- [x] **ダイソー：MAWA風すべらないハンガー** ✓
  - 購入本数：12本
  - 用途：シャツ、ボトムスなど
  - メモ：薄型PVC加工タイプ

### 既存のもので使えるもの
- ボックス/引き出し（下着・靴下・タオル用）

## 実装手順
1. [ ] ダイソーでハンガー購入
2. [ ] 洗濯物をハンガーと仕分けボックスに振り分け
3. [ ] クローゼット・引き出しに収納
4. [ ] 今後は干す時点でハンガー使用（畳む工程を最初から排除）

## メモ
- 靴下は全部同じ（ユニクロ無地黒）なのでペア探し不要
- タオルも畳まなくてOK
