# Fashion Project

Discordの#fashionチャンネルでの雑談内容を記録し、ファッションに関する知識を集約するプロジェクトです。

---

## 🎯 目的

1. **定番アイテムのまとめ** - アメカジ・ミリタリー・ワークウェアの基本知識
2. **購入履歴の記録** - いつ・何を・どこで買ったか詳細に管理
3. **ファッションのヒント集** - YouTube研究やスタイリングのコツを集約
4. **会話ログの保存** - ファッションについての雑談・発見を記録

## 📝 活用方針

### 新しいアイテムを購入したら
1. `purchases/history.md` に記録（日付, 価格, サイズ, 満足度など）
2. `ideas/wishlist.md` のステータスを更新
3. 着用感や満足度を後日追記

### YouTube で学んだら
- `tips/youtube-research.md` に追記
- 参考チャンネル名、動画タイトル、学んだポイント

### コーディネートの成功例
- `tips/styling-basics.md` または `tips/body-type-guide.md` に追記
- 写真があれば保存

### 日々の会話
- `conversations/YYYY-MM-DD.md` に日付ごとに記録
- ファッションに関する気づき、発見、悩みなど

### 定番アイテムの追加
- `items/` 内の各ファイルに追記
- 新カテゴリーが必要なら新ファイル作成（帽子、バッグなど）

## 🌟 プロジェクトの軸

### 1. 憧れの実現
- 中学時代の同級生のN-3B
- トップガンのG-1
- 台湾学生リーダーのM65
- → 「あの頃欲しかったもの」を今手に入れる

### 2. ストーリー重視
- ブランドヒエラルキーより「意味」
- 無名ブランドでも満足（思い出があれば）
- アルファは当時の憧れそのもの

### 3. 清潔感とのバランス
- ヴィンテージの無骨さ + 中年男性の清潔感
- きれいめアイテムとの組み合わせ
- コンディション・サイズ感・手入れが鍵

### 4. 体型に合わせる
- タイトはNG、ゆとりあるデザインが味方
- サイズ感が命
- 試行錯誤を楽しむ
